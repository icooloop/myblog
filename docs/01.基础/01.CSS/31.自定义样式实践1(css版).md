# 自定义样式实践(无框架css版)
## 1.样式结构示例
```bash
-main  # 存放公用型样式
  --base.css   # 基础样式全局引用
  --common.css  # 通用型公共样式
  --animate.css # 动画样式
-about #about模块
  --about.css # 局部公共样式
  --about1.css # 独立页面样式
  --about2.css # 独立页面样式
-activity #活动 模块
  --activity1.css # 独立页面样式
  --activity2.css # 独立页面样式
```
